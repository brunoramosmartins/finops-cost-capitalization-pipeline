# Accounting Policy Notes

This repository does not encode authoritative accounting policy in Phase 1.
Instead, it prepares the synthetic inputs that Gold models will later use to
generate accounting recommendations.

Phase 1 fields relevant to later policy evaluation:

- `capitalization_candidate`
- `initiative_id`
- `initiative_stage`
- `asset_lifecycle`
- `accounting_policy_version`

Phase 2 will formalize the first rule-based recommendation layer.
